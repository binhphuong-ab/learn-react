# KaizenNote Feature Guide (For Users)

## 1) What is KaizenNote?

KaizenNote helps you plan your week in hourly blocks, track activities/projects, write daily reflections, and review reports.

---

## 2) Getting Started

1. Open the app and create an account on the Register page.
2. Login with your username and password.
3. You will be redirected to the Planner.

---

## 3) Main Navigation

You have 4 tabs:

1. `Planner`: weekly schedule + daily reflection
2. `Dashboard`: charts and report summaries
3. `Settings`: manage activity types and projects
4. `Profile`: view username and logout

---

## 4) Planner: Weekly Hour Grid

The planner grid shows:

- 7 days (Monday to Sunday)
- 24 hours per day (00:00 to 23:00)
- Each box is one hour

How to use:

1. Click any hour box to select it.
2. Press `Enter` to open the edit popup.
3. Fill task note, activity type, and optional project.
4. Save.

Keyboard support in grid:

1. `Arrow Up/Down/Left/Right`: move selected box
2. `Enter`: open popup
3. `Delete` or `Backspace`: clear selected box task
4. `Command+C` (Mac) / `Ctrl+C` (Windows): copy selected box task
5. `Command+V` / `Ctrl+V`: paste to selected box and save

Notes:

- If selected box moves out of view, scroll follows automatically.
- Copy only works when selected box already has a task.

---

## 5) Task Popup Shortcuts

When popup is open:

1. `Enter`: Save task
2. `Delete` or `Backspace`: Delete task
3. `Esc`: Close popup

Important behavior:

- If you open popup on an empty box and press Enter without editing anything, the box stays empty.

---

## 6) Daily Reflection (3-Day Focus)

The reflection section shows 3 cards:

1. Yesterday
2. Focus day (center)
3. Tomorrow

You can:

1. Write `Plan`, `Result`, `Improvement`
2. Save each day independently
3. Move focus with `Previous day`, `Today`, `Next day`

---

## 7) Dashboard Reports

Dashboard includes:

1. Time by Activity
2. Time by Project
3. Weekly Day Totals chart
4. 8-Week Trends chart

How to use:

1. Pick a start and end date.
2. Click `Refresh reports`.
3. Review totals and trends.

---

## 8) Settings

In `Settings` tab you can manage:

1. Activity Types (name + color)
2. Projects (name + description + color)

Rules:

- You cannot delete an activity type or project if tasks are still using it.

---

## 9) Profile

In `Profile` tab:

1. See your username
2. Logout

---

## 10) Tips

1. Use keyboard to plan faster (arrows + enter + copy/paste).
2. Use consistent activity colors for easier report reading.
3. Keep daily reflection short and honest for better weekly review.
